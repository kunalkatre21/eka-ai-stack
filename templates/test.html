Console Commands

ComponentSystem.enableDebug() - Enable debug mode
debugComponents() - Show component information
ComponentSystem.getDebugInfo() - Get detailed debug info
console.table(ComponentSystem.getDebugInfo()) - Show as table

Usage Examples

Add Navigation with FA Pro:

<div data-component="navigation" data-config='{"logoIcon":"fad fa-hexagon-vertical-nft","items":[{"label":"Home","icon":"fal fa-home"}]}'></div>

Add Hero Section with Badge:

<div data-component="hero-section" data-config='{"badge":"New Feature","badgeIcon":"fad fa-sparkles","backgroundIcon":"fad fa-hexagon-vertical-nft"}'></div>

Add Card with Duotone Icons:

<div data-component="card" data-config='{"icon":"fad fa-card-heart","titleIcon":"fal fa-star","variant":"featured"}'></div>
